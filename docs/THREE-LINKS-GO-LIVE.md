## One-Time Setup Checklist

Do these steps once, in order. We'll supply the values where marked **[we provide]**.

1. **Fix the custom domain SSL** (5 min)  
   Go to Vercel Dashboard → Domains → centaurrobotics.com → renew the SSL certificate.

2. **Add three secret keys in Vercel** (10 min)  
   Vercel Dashboard → Settings → Environment Variables. Add:
   - `SUPABASE_SERVICE_ROLE_KEY` **[we provide]**
   - `RESEND_API_KEY` **[we provide]**
   - `NEXT_PUBLIC_SITE_URL` = `https://centaurrobotics.com` (or backup: `https://centaurrobotics1.vercel.app`)

3. **Run the database setup** (5 min, one time)  
   Log into Supabase → SQL Editor → paste and run `supabase/build-schema.sql`. This creates the tables for saved builds and enquiries.

4. **Create staff login accounts** (ongoing, 2 min per person)  
   In Supabase, create user accounts for each team member. They log in with email + password at `/staff`.  
   *Optional:* Set `STAFF_ALLOWED_EMAILS` in Vercel to restrict login to your domain (e.g., only @centaurrobotics.com).

5. **Verify the email sender in Resend** (10 min)  
   Resend Dashboard → Domains → add and verify `hello@centaurrobotics.com`. This lets the system send build confirmations and enquiry receipts.
