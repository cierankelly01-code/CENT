# Parked ideas (not part of the Centaur website)

## AI Operations Layer (Cieran's concept) — separate product, do NOT build into the Centaur site
A fully custom Claude-Code-built operations interface for companies. An "Obsidian-killer"
for AI-operated teams:
- Departments, human/agent teams, SOPs, tools and software all modelled as linked notes.
- Click any node to see what a department is, the SOPs it's connected to, the tools it uses.
- Navigable connection graph between all of it.
- Permission structures (e.g. can an employee/agent using tool X access Y?).
- Wrapped in a custom Claude chat interface where SOPs/tools/employees can be @-referenced
  inline and show up in the chat as live entities.
- Wedge: fully custom in ~a week vs. fighting Obsidian's limits.

**Decision:** strong concept, pursue as its own product. Keep entirely separate from the
Centaur marketing site (different audience/purpose). Revisit when the Centaur build is shipped.
